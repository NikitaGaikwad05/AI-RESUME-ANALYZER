## 💡 **ANALYSIS ENGINE DETAILS**

### **Scoring Algorithm:**

```javascript
Overall Score = (
    Keyword Match (40%) +
    Formatting Quality (25%) +
    Section Structure (20%) +
    Length & Readability (15%)
) / 100
```

### **Keyword Match (40%):**
- Detects: Frontend, Backend, Database, DevOps, Tools skills
- Matches: Job-role specific keywords
- Calculates: Found / Total keywords
- Returns: 0-100 percentage

### **Formatting Quality (25%):**
- Checks: Email/phone presence
- Checks: Bullet point usage
- Checks: Text length
- Returns: 0-100 score

### **Section Structure (20%):**
- Detects: Professional Summary
- Detects: Experience section
- Detects: Skills section
- Detects: Education section
- Returns: 0-100 score

### **Length & Readability (15%):**
- Optimal: 1000-2000 characters
- Too short: < 500 (40%)
- Too long: > 3000 (90%)
- Just right: 1000-2000 (95-100%)
