## 🛠️ **TROUBLESHOOTING**

### **Issue: Page doesn't load**
**Solutions:**
- Verify all 3 files exist (index.html, style.css, script.js)
- Try different browser (Chrome, Firefox)
- Clear browser cache (Ctrl+Shift+Delete)
- Check file paths are correct

### **Issue: Styles look broken**
**Solutions:**
- Refresh page (Ctrl+F5)
- Check style.css file size (should be 700+ KB)
- Try incognito/private mode
- Restart browser

### **Issue: Analysis button disabled**
**Solutions:**
- Upload a file first
- Type a job role
- Both fields must be filled

### **Issue: File upload fails**
**Solutions:**
- Check file size < 5MB
- Use supported format (PDF, DOCX, TXT)
- Try .txt file first for testing

### **Issue: Dark mode not working**
**Solutions:**
- Click button again to toggle
- Clear browser cache
- Check browser console (F12)
- Try different browser

### **Issue: History not saving**
**Solutions:**
- Check if localStorage is enabled
- Not in private/incognito mode
- Try regular mode
- Check storage not full
